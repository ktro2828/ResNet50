# ResNet50
ResNet50 with dropout module
- default drop-rate : 0.3
- default dataset : CIFAR10

- parser description:
  - --epoch       : the number of epoch (defult=30)
  - --batch       : batch size (default=128)
  - --weight-file : weight file if you did pretrain
  - --token       : Line token
